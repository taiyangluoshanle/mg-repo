---
title: PriceRange
description: 价格区间展示：最低价与最高价相等时显示单一价格，否则显示「min - max」。
---

## 安装

```bash
pnpm add @mg/ui-commerce
```

## 基础用法

<ComponentPreview>
  <PriceRangeBasicDemo />
</ComponentPreview>

```tsx
import { PriceRange } from '@mg/ui-commerce';

export default function Demo() {
  return <PriceRange min={49} max={99} currency="¥" />;
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `min` | `number` | — | 区间最低价 |
| `max` | `number` | — | 区间最高价 |
| `currency` | `string` | — | 货币符号或代码，参与格式化输出 |

除上表外，组件继承根 `span` 元素的 HTML 属性（如 `className`）。
