---
title: OrderCard
description: 订单卡片：订单号、状态徽章、商品行列表（图、标题、单价、数量）与合计。
---

## 安装

```bash
pnpm add @mg/ui-commerce
```

## 基础用法

```tsx
import { OrderCard } from '@mg/ui-commerce';

export default function Demo() {
  return (
    <OrderCard
      orderNo="20260402120001"
      status="待发货"
      statusVariant="warning"
      items={[
        {
          image: 'https://example.com/a.jpg',
          title: '商品 A',
          price: 59,
          quantity: 1,
        },
        {
          image: 'https://example.com/b.jpg',
          title: '商品 B',
          price: 39,
          quantity: 2,
        },
      ]}
      total={137}
    />
  );
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `orderNo` | `string` | — | 订单号展示文案 |
| `status` | `string` | — | 状态展示文案（由业务自定义） |
| `statusVariant` | `"success" \| "warning" \| "error" \| "default"` | — | 状态徽章配色映射 |
| `items` | `Array<{ image: string; title: string; price: number; quantity: number }>` | — | 订单行商品列表 |
| `total` | `number` | — | 订单合计金额（内部以 `¥` 格式化） |

除上表外，组件继承根元素的 HTML 属性（如 `className`）。
