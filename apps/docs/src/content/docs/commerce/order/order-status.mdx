---
title: OrderStatus
description: 订单状态徽章，将标准状态枚举映射为中文标签与 `Badge` 变体。
---

## 安装

```bash
pnpm add @mg/ui-commerce
```

## 基础用法

```tsx
import { OrderStatus } from '@mg/ui-commerce';

export default function Demo() {
  return <OrderStatus status="shipped" />;
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `status` | `"pending" \| "paid" \| "shipped" \| "delivered" \| "cancelled"` | — | 订单状态：`pending` 待付款、`paid` 已付款、`shipped` 配送中、`delivered` 已完成、`cancelled` 已取消 |

除上表外，组件继承根 `Badge` 外层可透传的 HTML 属性（如 `className`）。
