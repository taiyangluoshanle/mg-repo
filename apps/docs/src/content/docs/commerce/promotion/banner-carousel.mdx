---
title: BannerCarousel
description: 营销横幅轮播，支持多张图片、可选链接、自动播放与指示点/左右切换。
---

## 安装

```bash
pnpm add @mg/ui-commerce
```

## 基础用法

```tsx
import { BannerCarousel } from '@mg/ui-commerce';

export default function Demo() {
  return (
    <BannerCarousel
      items={[
        { image: 'https://example.com/b1.jpg', alt: '活动一', href: '/promo/1' },
        { image: 'https://example.com/b2.jpg', alt: '活动二' },
      ]}
      autoPlay
      interval={5000}
    />
  );
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `items` | `Array<{ image: string; alt: string; href?: string }>` | — | 轮播项；`href` 存在时覆盖整张当前图的可点击链接（无障碍仅读 `alt`） |
| `autoPlay` | `boolean` | `true` | 是否自动轮播；仅当项数大于 1 时生效 |
| `interval` | `number` | `5000` | 自动切换间隔（毫秒） |

除上表外，组件继承根元素的 HTML 属性（如 `className`）。该组件需在客户端环境中使用。
