---
title: CountdownTimer
description: 活动倒计时，按天/时/分/秒展示剩余时间，结束时可选触发 `onEnd`（仅触发一次）。
---

## 安装

```bash
pnpm add @mg/ui-commerce
```

## 基础用法

<ComponentPreview>
  <CountdownTimerBasicDemo />
</ComponentPreview>

```tsx
import { CountdownTimer } from '@mg/ui-commerce';

export default function Demo() {
  const end = new Date(Date.now() + 2 * 60 * 60 * 1000);

  return (
    <CountdownTimer
      endTime={end}
      onEnd={() => console.log('ended')}
    />
  );
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `endTime` | `Date \| string \| number` | — | 结束时刻；`number` 为毫秒时间戳，若小于 `1e12` 会按秒时间戳处理 |
| `onEnd` | `() => void` | — | 倒计时归零时回调，同一轮倒计时内仅调用一次 |

除上表外，组件继承根元素的 HTML 属性（如 `className`）。该组件需在客户端环境中使用。
