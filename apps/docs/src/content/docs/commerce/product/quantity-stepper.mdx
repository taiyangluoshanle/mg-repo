---
title: QuantityStepper
description: 数量步进器，提供减号、数字输入与加号，并在 `min`/`max` 范围内约束数量。
---

## 安装

```bash
pnpm add @mg/ui-commerce
```

## 基础用法

<ComponentPreview>
  <QuantityStepperBasicDemo />
</ComponentPreview>

```tsx
import { useState } from 'react';
import { QuantityStepper } from '@mg/ui-commerce';

export default function Demo() {
  const [value, setValue] = useState(1);

  return (
    <QuantityStepper
      value={value}
      onValueChange={setValue}
      min={1}
      max={99}
    />
  );
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `value` | `number` | — | 当前数量 |
| `onValueChange` | `(next: number) => void` | — | 数量变化回调 |
| `min` | `number` | `1` | 允许的最小值 |
| `max` | `number` | `99` | 允许的最大值 |

除上表外，组件继承根元素的 HTML 属性（如 `className`）。该组件需在客户端环境中使用。
