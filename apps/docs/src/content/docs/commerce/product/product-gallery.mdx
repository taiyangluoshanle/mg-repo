---
title: ProductGallery
description: 商品图库：上方大图展示当前选中图片，下方为可切换的缩略图列表。
---

## 安装

```bash
pnpm add @mg/ui-commerce
```

## 基础用法

<ComponentPreview>
  <ProductGalleryBasicDemo />
</ComponentPreview>

```tsx
import { ProductGallery } from '@mg/ui-commerce';

export default function Demo() {
  return (
    <ProductGallery
      images={[
        'https://example.com/a.jpg',
        'https://example.com/b.jpg',
        'https://example.com/c.jpg',
      ]}
      alt="商品主图"
    />
  );
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `images` | `string[]` | — | 图片 URL 列表；多于一张时显示缩略图行 |
| `alt` | `string` | — | 主图与无障碍相关的替代文本（缩略图使用衍生 `aria-label`） |

除上表外，组件继承根元素的 HTML 属性（如 `className`）。该组件需在客户端环境中使用（内部使用 `useState`）。
