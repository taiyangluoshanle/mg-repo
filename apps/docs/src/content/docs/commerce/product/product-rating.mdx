---
title: ProductRating
description: 商品评分展示，以半星精度渲染 0–5 星，并显示评价条数（中文本地化格式）。
---

## 安装

```bash
pnpm add @mg/ui-commerce
```

## 基础用法

```tsx
import { ProductRating } from '@mg/ui-commerce';

export default function Demo() {
  return <ProductRating value={4.2} count={1280} />;
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `value` | `number` | — | 评分，范围 0–5，支持小数（半星填充） |
| `count` | `number` | — | 评价总数，用于展示「N 条评价」 |

除上表外，组件继承根元素的 HTML 属性（如 `className`）。
