---
title: CartDrawer
description: 右侧滑出的购物车抽屉容器，基于 `@mg/ui` 的 `Drawer`，控制开关并包裹子内容。
---

## 安装

```bash
pnpm add @mg/ui-commerce
```

## 基础用法

```tsx
import { useState } from 'react';
import { CartDrawer } from '@mg/ui-commerce';

export default function Demo() {
  const [open, setOpen] = useState(false);

  return (
    <CartDrawer open={open} onOpenChange={setOpen}>
      <div className="p-4">购物车内容</div>
    </CartDrawer>
  );
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `open` | `boolean` | — | 是否打开抽屉 |
| `onOpenChange` | `(open: boolean) => void` | — | 开关状态变化回调 |
| `children` | `React.ReactNode` | — | 抽屉主体内容 |

除上表外，组件继承外层包装元素的 HTML 属性（如 `className`）。该组件需在客户端环境中使用。
