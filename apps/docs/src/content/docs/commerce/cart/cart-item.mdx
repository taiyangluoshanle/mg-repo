---
title: CartItem
description: 购物车行：缩略图、标题、SKU 文案、内置数量步进器、小计与删除操作。
---

## 安装

```bash
pnpm add @mg/ui-commerce
```

## 基础用法

```tsx
import { CartItem } from '@mg/ui-commerce';

export default function Demo() {
  return (
    <CartItem
      image="https://example.com/item.jpg"
      title="示例商品"
      price={99}
      quantity={2}
      sku="颜色：黑 / 尺码：M"
      onQuantityChange={(n) => console.log('qty', n)}
      onRemove={() => console.log('remove')}
    />
  );
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `image` | `string` | — | 商品缩略图 URL |
| `title` | `string` | — | 商品标题 |
| `price` | `number` | — | 单价（内部按人民币符号格式化行小计） |
| `quantity` | `number` | — | 当前数量 |
| `sku` | `string` | — | SKU 或规格摘要文案 |
| `onQuantityChange` | `(next: number) => void` | — | 数量变更回调 |
| `onRemove` | `() => void` | — | 点击「删除」时回调 |

除上表外，组件继承根元素的 HTML 属性（如 `className`）。
