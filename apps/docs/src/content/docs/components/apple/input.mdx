## 变体 (Variants)

支持三种变体：`default`（标准边框）、`search`（搜索栏风格）、`underline`（下划线风格）。

<ComponentBlock
  cli="@mg/components"
  code={`import { Input } from "@mg/components";

export function Demo() {
  return (
    <div className="flex flex-col gap-5">
      <Input variant="default" placeholder="Apple ID" />
      <Input variant="search" placeholder="搜索 Apple 产品..." startAdornment={<SearchIcon />} />
      <Input variant="underline" placeholder="您的姓名" />
    </div>
  );
}`}
>
  <AppleInputVariantsDemo />
</ComponentBlock>

## 尺寸 (Sizes)

支持 `sm`（36px）、`md`（44px）、`lg`（52px）三种尺寸。

<ComponentBlock
  cli="@mg/components"
  code={`import { Input } from "@mg/components";

export function Demo() {
  return (
    <div className="flex flex-col gap-5">
      <Input size="sm" placeholder="Small — 36px" />
      <Input size="md" placeholder="Medium — 44px" />
      <Input size="lg" placeholder="Large — 52px" />
    </div>
  );
}`}
>
  <AppleInputSizesDemo />
</ComponentBlock>

## 前后缀装饰 (Adornments)

可通过 `startAdornment` 和 `endAdornment` 添加图标等装饰元素。

<ComponentBlock
  cli="@mg/components"
  code={`import { Input } from "@mg/components";

export function Demo() {
  return (
    <div className="flex flex-col gap-5">
      <Input variant="search" placeholder="搜索..." startAdornment={<SearchIcon />} />
      <Input variant="default" placeholder="you@apple.com" startAdornment={<MailIcon />} />
    </div>
  );
}`}
>
  <AppleInputAdornmentDemo />
</ComponentBlock>

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `variant` | `"default" \| "search" \| "underline"` | `"default"` | 输入框变体 |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | 输入框尺寸 |
| `startAdornment` | `React.ReactNode` | `-` | 左侧装饰（如搜索图标） |
| `endAdornment` | `React.ReactNode` | `-` | 右侧装饰 |
| `disabled` | `boolean` | `false` | 禁用状态 |
| `className` | `string` | `-` | 附加到 input 的类名 |
