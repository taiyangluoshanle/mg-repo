---
title: Menubar
description: 应用级菜单栏，包含多个下拉菜单。基于 Base UI Menubar 构建。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <MenubarBasicDemo />
</ComponentPreview>

```tsx
import { Menubar, DropdownMenu, DropdownMenuTrigger, DropdownMenuContent, DropdownMenuItem } from "@mg/ui";

export default function Demo() {
  return (
    <Menubar>
      <DropdownMenu>
        <DropdownMenuTrigger>文件</DropdownMenuTrigger>
        <DropdownMenuContent>
          <DropdownMenuItem>新建</DropdownMenuItem>
          <DropdownMenuItem>打开</DropdownMenuItem>
        </DropdownMenuContent>
      </DropdownMenu>
    </Menubar>
  );
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | 方向 |
| `disabled` | `boolean` | `false` | 禁用 |
| `loopFocus` | `boolean` | `true` | 键盘焦点循环 |
| `modal` | `boolean` | `true` | 模态模式 |
