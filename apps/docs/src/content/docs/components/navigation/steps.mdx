---
title: Steps
description: 步骤条（Stepper）组件，按顺序展示流程阶段，支持标题与可选描述文案。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

```tsx
import { Steps } from '@mg/ui';

const items = [
  { title: '填写信息', description: '基础资料' },
  { title: '确认订单' },
  { title: '完成支付', description: '安全支付' },
];

export default function Demo() {
  return (
    <Steps
      current={1}
      items={items}
    />
  );
}
```

## API

### Steps

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `current` | `number` | — | **必填**，当前步骤索引（从 0 开始）；该索引对应步骤为高亮，之前步骤为已完成 |
| `items` | `Array<{ title: string; description?: string }>` | — | **必填**，步骤列表；`title` 为标题，`description` 为可选副标题 |
| `className` | `string` | — | 样式类名 |
| 其他 | `React.HTMLAttributes<HTMLOListElement>` | — | 透传至外层 `ol` |
