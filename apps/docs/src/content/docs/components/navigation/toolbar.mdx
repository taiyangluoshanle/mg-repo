---
title: Toolbar
description: 一组按钮和控件的容器，管理键盘导航（roving tabindex）。基于 Base UI Toolbar 构建。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <ToolbarBasicDemo />
</ComponentPreview>

```tsx
import { ToolbarRoot, ToolbarGroup, ToolbarButton, ToolbarSeparator, ToolbarLink } from "@mg/ui";

export default function Demo() {
  return (
    <ToolbarRoot>
      <ToolbarGroup>
        <ToolbarButton aria-label="加粗">B</ToolbarButton>
        <ToolbarButton aria-label="斜体">I</ToolbarButton>
      </ToolbarGroup>
      <ToolbarSeparator />
      <ToolbarLink href="#">帮助</ToolbarLink>
    </ToolbarRoot>
  );
}
```

## API

### ToolbarRoot

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | 方向 |
| `disabled` | `boolean` | `false` | 禁用全部按钮 |
| `loopFocus` | `boolean` | `true` | 键盘焦点循环 |
