---
title: Pagination
description: 分页导航组件，根据当前页与总页数渲染页码、省略号及上一页/下一页按钮。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <PaginationBasicDemo />
</ComponentPreview>

```tsx
import { Pagination } from '@mg/ui';
import { useState } from 'react';

export default function Demo() {
  const [page, setPage] = useState(1);

  return (
    <Pagination
      currentPage={page}
      totalPages={12}
      onPageChange={setPage}
    />
  );
}
```

## API

### Pagination

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `currentPage` | `number` | — | **必填**，当前页码（从 1 开始） |
| `totalPages` | `number` | — | **必填**，总页数；小于 1 时不渲染 |
| `onPageChange` | `(page: number) => void` | — | **必填**，页码变更回调 |
| `className` | `string` | — | 样式类名 |
| 其他 | `React.ComponentPropsWithoutRef<'nav'>` | — | 透传至 `nav`（`aria-label="Pagination"`） |
