---
title: Switch
description: 开关按钮，支持受控/非受控与三种尺寸，语义为 `role="switch"`。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <SwitchBasicDemo />
</ComponentPreview>

```tsx
import { Switch } from '@mg/ui';

export default function Demo() {
  return (
    <Switch
      checked={false}
      onCheckedChange={(on) => console.log(on)}
      size="md"
    />
  );
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | 轨道与滑块尺寸 |
| `checked` | `boolean` | `-` | 受控开启状态 |
| `defaultChecked` | `boolean` | `false`（内部初始） | 非受控初始开启 |
| `onCheckedChange` | `(checked: boolean) => void` | `-` | 点击切换后的状态 |
| `disabled` | `boolean` | `-` | 禁用 |
| `type` | `React.HTMLButtonElement["type"]` | `"button"` | 按钮 `type`，避免表单误提交 |
| `className` | `string` | `-` | 附加到轨道 `button` |
| 其他 | `Omit<React.ComponentPropsWithoutRef<"button">, "onChange">` | `-` | 继承原生 `button` 属性 |
