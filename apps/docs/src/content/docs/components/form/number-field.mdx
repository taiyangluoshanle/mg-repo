---
title: Number Field
description: 带递增/递减按钮的数字输入框。基于 Base UI NumberField 构建。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <NumberFieldBasicDemo />
</ComponentPreview>

```tsx
import {
  NumberFieldRoot,
  NumberFieldGroup,
  NumberFieldInput,
  NumberFieldIncrement,
  NumberFieldDecrement,
} from '@mg/ui';

export default function Demo() {
  return (
    <NumberFieldRoot defaultValue={1} min={0} max={99}>
      <NumberFieldGroup>
        <NumberFieldDecrement>−</NumberFieldDecrement>
        <NumberFieldInput />
        <NumberFieldIncrement>+</NumberFieldIncrement>
      </NumberFieldGroup>
    </NumberFieldRoot>
  );
}
```

## API

### NumberFieldRoot

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `value` | `number` | `-` | 受控数值 |
| `defaultValue` | `number` | `-` | 非受控初始值 |
| `onValueChange` | `(value, eventDetails) => void` | `-` | 数值变化回调 |
| `min` | `number` | `-` | 最小值 |
| `max` | `number` | `-` | 最大值 |
| `step` | `number` | `1` | 步进值 |
| `disabled` | `boolean` | `false` | 是否禁用 |
