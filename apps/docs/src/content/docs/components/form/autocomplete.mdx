---
title: Autocomplete
description: 输入时提供建议的自动补全组件。基于 Base UI Autocomplete 构建，使用 Combobox 子组件。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <AutocompleteBasicDemo />
</ComponentPreview>

```tsx
import {
  AutocompleteRoot,
  ComboboxInput,
  ComboboxPortal,
  ComboboxPositioner,
  ComboboxPopup,
  ComboboxList,
  ComboboxItem,
} from '@mg/ui';

const cities = ['北京', '上海', '广州'];

export default function Demo() {
  return (
    <AutocompleteRoot>
      <ComboboxInput placeholder="输入城市名..." />
      <ComboboxPortal>
        <ComboboxPositioner sideOffset={4}>
          <ComboboxPopup>
            <ComboboxList>
              {cities.map((city) => (
                <ComboboxItem key={city} value={city}>
                  {city}
                </ComboboxItem>
              ))}
            </ComboboxList>
          </ComboboxPopup>
        </ComboboxPositioner>
      </ComboboxPortal>
    </AutocompleteRoot>
  );
}
```

## API

### AutocompleteRoot

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `value` | `string \| null` | `-` | 受控输入值 |
| `onValueChange` | `(value, eventDetails) => void` | `-` | 输入值变化回调 |
| `disabled` | `boolean` | `false` | 是否禁用 |

列表、输入、弹层等子组件与 Combobox 一致，使用 `ComboboxInput`、`ComboboxList`、`ComboboxItem` 等组合即可。
