---
title: Fieldset
description: 原生 fieldset 元素的样式化封装，带可定制的 legend。基于 Base UI Fieldset 构建。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <FieldsetBasicDemo />
</ComponentPreview>

```tsx
import { FieldsetRoot, FieldsetLegend, Input } from '@mg/ui';

export default function Demo() {
  return (
    <FieldsetRoot>
      <FieldsetLegend>个人信息</FieldsetLegend>
      <div className="space-y-3">
        <Input placeholder="姓名" />
        <Input placeholder="邮箱" type="email" />
      </div>
    </FieldsetRoot>
  );
}
```

## API

### FieldsetRoot

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `disabled` | `boolean` | `false` | 是否禁用组内控件 |
| `className` | `string` | `-` | 根元素类名 |

### FieldsetLegend

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `className` | `string` | `-` | 图例类名 |
