---
title: Select
description: 原生下拉选择器，与 Input 保持一致的视觉风格。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <SelectBasicDemo />
</ComponentPreview>

```tsx
import { Select } from '@mg/ui';

export default function Demo() {
  return (
    <Select placeholder="请选择">
      <option value="1">选项一</option>
      <option value="2">选项二</option>
      <option value="3">选项三</option>
    </Select>
  );
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `variant` | `"default" \| "error"` | `"default"` | 变体 |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | 尺寸 |
| `placeholder` | `string` | `-` | 占位文本（渲染为禁用首选项） |
