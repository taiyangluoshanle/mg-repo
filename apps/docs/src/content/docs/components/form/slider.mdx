---
title: Slider
description: 基于原生 `input[type=range]` 的范围滑块，可配置最小值、最大值、步长与数值回调。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <SliderBasicDemo />
</ComponentPreview>

```tsx
import { Slider } from '@mg/ui';

export default function Demo() {
  return (
    <Slider
      min={0}
      max={100}
      step={1}
      defaultValue={50}
      onValueChange={(v) => console.log(v)}
    />
  );
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `min` | `number` | `0` | 最小值 |
| `max` | `number` | `100` | 最大值 |
| `step` | `number` | `1` | 步长 |
| `value` | `number` | `-` | 受控当前值 |
| `defaultValue` | `number` | `-` | 非受控初始值 |
| `onValueChange` | `(value: number) => void` | `-` | 拖拽或键盘改变后的数值；内部仍会调用原生 `onChange` |
| `disabled` | `boolean` | `-` | 禁用 |
| `className` | `string` | `-` | 附加类名 |
| 其他 | `Omit<React.ComponentPropsWithoutRef<"input">, "type" \| "size">` | `-` | 继承原生 `range` 输入属性 |
