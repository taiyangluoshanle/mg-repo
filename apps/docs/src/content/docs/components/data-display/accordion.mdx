---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <AccordionBasicDemo />
</ComponentPreview>

```tsx
import { Accordion, AccordionItem } from '@mg/ui';

export default function Demo() {
  return (
    <Accordion>
      <AccordionItem title="第一条标题" defaultOpen>
        第一条的详细内容。
      </AccordionItem>
      <AccordionItem title="第二条标题">
        默认收起，点击标题可展开。
      </AccordionItem>
    </Accordion>
  );
}
```

## API

### Accordion

| 属性 | 类型 | 说明 |
|------|------|------|
| `className` | `string` | 附加类名 |
| 其他 | `HTMLAttributes<HTMLDivElement>` | 包裹所有子项的容器 |

### AccordionItem

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `title` | `ReactNode` | （必填） | 触发区域展示的标题 |
| `defaultOpen` | `boolean` | `false` | 初始是否展开 |
| `children` | `ReactNode` | — | 展开后可见的内容 |
| `className` | `string` | — | 单项外层类名 |
| 其他 | `HTMLAttributes<HTMLDivElement>`（不含原生 `title`） | — | 单项容器属性 |
