---
title: Badge
description: 用于展示状态、分类或计数的紧凑型标签，支持多种语义色与尺寸。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <BadgeVariantsDemo />
</ComponentPreview>

```tsx
import { Badge } from '@mg/ui';

export default function Demo() {
  return (
    <div className="flex flex-wrap gap-2">
      <Badge variant="default">默认</Badge>
      <Badge variant="secondary">次要</Badge>
      <Badge variant="success">成功</Badge>
      <Badge variant="warning">警告</Badge>
      <Badge variant="error">错误</Badge>
      <Badge variant="outline">描边</Badge>
      <Badge size="sm">小尺寸</Badge>
      <Badge size="md">中尺寸</Badge>
    </div>
  );
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `variant` | `"default" \| "secondary" \| "success" \| "warning" \| "error" \| "outline"` | `"default"` | 视觉变体 |
| `size` | `"sm" \| "md"` | `"md"` | 尺寸 |
| `className` | `string` | — | 附加类名 |
| 其他 | `HTMLAttributes<HTMLSpanElement>` | — | 原生 `span` 属性（如 `children`） |
