---
title: Skeleton
description: 加载占位骨架屏，支持文本行、圆形与矩形等形状，可自定义宽高。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <SkeletonBasicDemo />
</ComponentPreview>

```tsx
import { Skeleton } from '@mg/ui';

export default function Demo() {
  return (
    <div className="flex flex-col gap-4 max-w-md">
      <Skeleton variant="text" />
      <Skeleton variant="rectangular" width="100%" height={120} />
      <div className="flex items-center gap-3">
        <Skeleton variant="circular" width={40} height={40} />
        <div className="flex-1 space-y-2">
          <Skeleton variant="text" />
          <Skeleton variant="text" className="w-2/3" />
        </div>
      </div>
    </div>
  );
}
```

`variant` 为 `circular` 或 `rectangular` 时需配合 `width`、`height` 控制尺寸；`text` 变体自带默认高度与全宽样式。

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `variant` | `"text" \| "circular" \| "rectangular"` | `"text"` | 骨架形状 |
| `width` | `CSSProperties["width"]` | — | 内联宽度 |
| `height` | `CSSProperties["height"]` | — | 内联高度 |
| `className` | `string` | — | 附加类名 |
| `style` | `CSSProperties` | — | 会与 `width`/`height` 合并 |
| 其他 | `HTMLAttributes<HTMLDivElement>` | — | 原生 `div` 属性 |
