---
title: List
description: 列表与列表项；支持有序列表，以及列表项前置图标。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <ListBasicDemo />
</ComponentPreview>

```tsx
import { List, ListItem } from '@mg/ui';

export default function Demo() {
  return (
    <div className="space-y-6">
      <List>
        <ListItem icon={<span>•</span>}>第一项</ListItem>
        <ListItem icon={<span>•</span>}>第二项</ListItem>
      </List>
      <List ordered>
        <ListItem>有序第一项</ListItem>
        <ListItem>有序第二项</ListItem>
      </List>
    </div>
  );
}
```

`ordered` 为 `true` 时渲染 `<ol>` 并显示数字序号；否则为 `<ul>` 且无默认项目符号，适合配合 `icon` 自定义前缀。

## API

### List

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `ordered` | `boolean` | — | 为 `true` 时使用有序列表 |
| `className` | `string` | — | 附加类名 |
| 其他 | `HTMLAttributes<HTMLUListElement>` | — | 列表根元素属性 |

### ListItem

| 属性 | 类型 | 说明 |
|------|------|------|
| `icon` | `ReactNode` | 可选，显示在文案左侧的图标或符号 |
| 其他 | `LiHTMLAttributes<HTMLLIElement>` | 原生 `li` 属性 |
