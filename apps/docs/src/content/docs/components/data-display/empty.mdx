---
title: Empty
description: 空状态占位，用于列表无数据、搜索无结果等场景；可配置图标、文案与操作区。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <EmptyBasicDemo />
</ComponentPreview>

```tsx
import { Button, Empty } from '@mg/ui';

export default function Demo() {
  return (
    <Empty
      icon={<span aria-hidden>📭</span>}
      title="暂无数据"
      description="可以尝试调整筛选条件，或稍后再试。"
      action={<Button variant="primary">新建</Button>}
    />
  );
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `title` | `string` | （必填） | 主标题 |
| `description` | `string` | — | 辅助说明 |
| `icon` | `ReactNode` | — | 顶部图标或插图 |
| `action` | `ReactNode` | — | 底部操作区（如按钮） |
| `className` | `string` | — | 附加类名 |
| 其他 | `HTMLAttributes<HTMLDivElement>` | — | 容器带有 `role="status"` |
