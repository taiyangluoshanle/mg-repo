---
title: Container
description: 居中、限制最大宽度并带水平内边距的页面级内容容器。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <ContainerBasicDemo />
</ComponentPreview>

```tsx
import { Container } from '@mg/ui';

export default function Demo() {
  return (
    <Container maxWidth="lg">
      示例内容
    </Container>
  );
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `maxWidth` | `"sm" \| "md" \| "lg" \| "xl" \| "2xl" \| "full"` | `"xl"` | 最大宽度断点，对应 Tailwind `max-w-*` |
| `className` | `string` | `-` | 附加类名 |
| 其他 | `React.HTMLAttributes<HTMLDivElement>` | `-` | 继承原生 `div` 属性 |
