---
title: AspectRatio
description: 按宽高比约束尺寸的容器，常用于图片或视频占位，默认 16:9。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <AspectRatioBasicDemo />
</ComponentPreview>

```tsx
import { AspectRatio } from '@mg/ui';

export default function Demo() {
  return (
    <AspectRatio ratio={16 / 9} className="rounded-md bg-muted">
      示例内容
    </AspectRatio>
  );
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `ratio` | `number` | `16 / 9` | 宽 ÷ 高，传给 CSS `aspect-ratio` |
| `className` | `string` | `-` | 附加类名 |
| 其他 | `React.HTMLAttributes<HTMLDivElement>` | `-` | 继承原生 `div` 属性；`style` 会与内部 `aspectRatio` 合并 |
