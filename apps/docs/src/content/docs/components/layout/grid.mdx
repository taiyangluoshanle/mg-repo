---
title: Grid
description: CSS Grid 布局，支持固定列数或响应式列数配置与间距。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <GridBasicDemo />
</ComponentPreview>

```tsx
import { Grid } from '@mg/ui';

export default function Demo() {
  return (
    <Grid cols={2} gap={4}>
      <div>单元 1</div>
      <div>单元 2</div>
    </Grid>
  );
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `cols` | `1 \| … \| 12` 或 `{ sm?: 1–12; md?: 1–12; lg?: 1–12 }` | （必填） | 列数；对象形式时在对应断点覆盖，基础列为 `grid-cols-1` |
| `gap` | `0–12` | `4` | 网格间距 |
| `className` | `string` | `-` | 附加类名 |
| 其他 | `React.HTMLAttributes<HTMLDivElement>` | `-` | 继承原生 `div` 属性 |
