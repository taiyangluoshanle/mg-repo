---
title: Stack
description: 基于 Flex 的堆叠布局：通用 Stack、纵向 VStack、横向 HStack，可配置间距与对齐。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <StackBasicDemo />
</ComponentPreview>

```tsx
import { VStack, HStack } from '@mg/ui';

export default function Demo() {
  return (
    <VStack gap={4} align="start">
      <span>第一项</span>
      <HStack gap={2} justify="center">
        <span>A</span>
        <span>B</span>
      </HStack>
    </VStack>
  );
}
```

## API

### Stack

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `direction` | `"row" \| "column"` | `"column"` | 主轴方向 |
| `gap` | `0–12` | `4` | 子项间距（Tailwind `gap-*` 刻度） |
| `align` | `"start" \| "center" \| "end" \| "stretch" \| "baseline"` | `"stretch"` | `align-items` |
| `justify` | `"start" \| "center" \| "end" \| "between" \| "around" \| "evenly"` | `"start"` | `justify-content` |
| `className` | `string` | `-` | 附加类名 |
| 其他 | `React.HTMLAttributes<HTMLDivElement>` | `-` | 继承原生 `div` 属性 |

### VStack / HStack

与 Stack 相同，但分别固定为 `direction="column"` 与 `direction="row"`，类型中不包含 `direction`。
