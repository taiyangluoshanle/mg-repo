---
title: ScrollToTop
description: 回到顶部按钮，向上滚动时自动出现。
---

## 基本用法

<ComponentBlock cli="@mg/ui-commerce" code={`import { ScrollToTop } from "@mg/ui-commerce";\n\n<ScrollToTop threshold={300} />`}>
  <ScrollToTopDemo />
</ComponentBlock>

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| threshold | `number` | `300` | 显示阈值（px） |
| smooth | `boolean` | `true` | 是否平滑滚动 |
| className | `string` | - | 自定义样式 |
