  <BoxCountdownDemo />

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| endTime | `Date \| number` | - | 结束时间 |
| bgColor | `string` | `"bg-[#FF5A1F]"` | 数字块背景色 |
| textColor | `string` | `"text-black"` | 单位文字颜色 |
| separatorColor | `string` | `"text-neutral-300"` | 分隔符颜色 |
| labels | `{ days, hours, minutes, seconds }` | 中文 | 时间单位文案 |
| children | `({ timer }) => ReactNode` | - | 自定义布局 |
| onEnd | `() => void` | - | 结束回调 |
