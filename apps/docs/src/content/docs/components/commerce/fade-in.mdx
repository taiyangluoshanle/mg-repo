## 基本用法

<ComponentBlock cli="@mg/ui-commerce" code={`import { FadeIn } from "@mg/ui-commerce";\n\n<FadeIn duration={1000} translateY="10px">\n  <div>滚动到此处时渐入显示</div>\n</FadeIn>`}>
  <FadeInDemo />
</ComponentBlock>

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| duration | `number` | `1000` | 动画时长（ms） |
| translateY | `string` | `"10px"` | 初始偏移量 |
| threshold | `number` | `0.1` | 可见比例阈值 |
| once | `boolean` | `true` | 是否只触发一次 |
