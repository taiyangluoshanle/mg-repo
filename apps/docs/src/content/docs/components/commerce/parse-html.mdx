## 基本用法

<ComponentBlock cli="@mg/ui-commerce" code={`import { ParseHtml } from "@mg/ui-commerce";\n\n<ParseHtml html='<p>这是 <strong>富文本</strong> 内容</p>' />`}>
  <ParseHtmlDemo />
</ComponentBlock>

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| html | `string` | - | HTML 字符串 |
| proseClassName | `string` | - | prose 样式覆盖 |
| className | `string` | - | 容器类名 |
