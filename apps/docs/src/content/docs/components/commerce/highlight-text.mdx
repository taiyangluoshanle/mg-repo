## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| text | `string` | - | 完整文本 |
| highlight | `string \| string[] \| RegExp \| null` | `null` | 要高亮的模式 |
| highlightColor | `string` | - | 高亮颜色（优先于 class） |
| highlightClassName | `string` | - | 高亮样式类名 |
| caseSensitive | `boolean` | `false` | 是否区分大小写 |
| tag | `keyof JSX.IntrinsicElements` | `"span"` | 外层标签 |
| onHighlightClick | `(value: string, index: number) => void` | - | 高亮项点击回调 |
