---
title: Collapse
description: 展开/收起组件，基于 CSS 高度过渡实现平滑动画。
---

## 基本用法

<ComponentBlock cli="@mg/ui-commerce" code={`import { Collapse } from "@mg/ui-commerce";\n\nconst [open, setOpen] = useState(false);\n\n<button onClick={() => setOpen(!open)}>切换</button>\n<Collapse open={open}>\n  <div>可展开收起的内容</div>\n</Collapse>`}>
  <CollapseDemo />
</ComponentBlock>

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| open | `boolean` | - | 是否展开 |
| duration | `number` | `300` | 动画时长（ms） |
| className | `string` | - | 容器类名 |
