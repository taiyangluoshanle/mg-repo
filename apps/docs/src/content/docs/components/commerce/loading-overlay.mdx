---
title: LoadingOverlay
description: 全屏加载遮罩组件，适用于页面级加载状态。
---

## 基本用法

<ComponentBlock cli="@mg/ui-commerce" code={`import { LoadingOverlay } from "@mg/ui-commerce";\n\n<LoadingOverlay open={loading} />`}>
  <LoadingOverlayDemo />
</ComponentBlock>

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| open | `boolean` | - | 是否显示 |
| spinnerClassName | `string` | - | 自定义 spinner 样式 |
| className | `string` | - | 遮罩层类名 |
