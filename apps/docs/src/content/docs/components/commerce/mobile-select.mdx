## 基本用法

<ComponentBlock cli="@mg/ui-commerce" code={`import { MobileSelect } from "@mg/ui-commerce";\n\nconst [value, setValue] = useState("");\n\n<MobileSelect\n  value={value}\n  onChange={setValue}\n  placeholder="选择产品型号"\n  options={[\n    { label: "EP2000", value: "ep2000" },\n    { label: "EP760", value: "ep760" },\n  ]}\n/>`}>
  <MobileSelectDemo />
</ComponentBlock>

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| value | `string` | - | 当前值 |
| onChange | `(value: string) => void` | - | 值变化回调 |
| options | `MobileSelectOption[]` | - | 选项列表 `{ label, value }` |
| placeholder | `string` | `"请选择"` | 占位文字 |
| disabled | `boolean` | `false` | 禁用 |
| breakpoint | `number` | `1024` | 移动端断点（px） |
| triggerClassName | `string` | - | 触发器类名 |
| optionClassName | `string` | - | 选项类名 |
