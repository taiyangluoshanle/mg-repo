title: ClientPagination

---

## 基本用法

<ComponentBlock cli="@mg/ui-commerce" code={`import { ClientPagination } from "@mg/ui-commerce";\n\nconst [page, setPage] = useState(1);\n\n<ClientPagination\n  current={page}\n  total={100}\n  pageSize={10}\n  onChange={setPage}\n/>`}>
  <ClientPaginationDemo />
</ComponentBlock>

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| current | `number` | - | 当前页码 |
| total | `number` | - | 总数据条数 |
| pageSize | `number` | `10` | 每页条数 |
| siblingCount | `number` | `1` | 当前页前后显示页数 |
| onChange | `(page: number) => void` | - | 页码变化回调 |
