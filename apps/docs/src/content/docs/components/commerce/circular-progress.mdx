---
title: CircularProgress
description: 圆形进度条组件，基于 SVG 实现。
---

## 基本用法

<ComponentBlock cli="@mg/ui-commerce" code={`import { CircularProgress } from "@mg/ui-commerce";\n\n<CircularProgress value={75} size={48} />`}>
  <CircularProgressDemo />
</ComponentBlock>

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| value | `number` | `0` | 进度值 0-100 |
| size | `number` | `32` | 尺寸（px） |
| strokeWidth | `number` | `4` | 线条宽度 |
| trackColor | `string` | `"stroke-black/30"` | 轨道颜色 |
| indicatorColor | `string` | `"stroke-blue-600"` | 指示器颜色 |
