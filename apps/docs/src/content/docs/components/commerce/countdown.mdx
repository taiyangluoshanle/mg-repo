---
title: Countdown
description: 倒计时组件，支持默认渲染和 Render Props 自定义渲染。
---

## 基本用法

<ComponentBlock cli="@mg/ui-commerce" code={`import { Countdown } from "@mg/ui-commerce";\n\n<Countdown pad endTime={Date.now() + 3 * 24 * 3600000} />\n\n// Render Props 自定义渲染\n<Countdown endTime={endTime}>\n  {({ formatTime }) => (\n    <div className="flex gap-3">\n      {formatTime.map(([value, unit]) => (\n        <div key={unit}>\n          <span>{value}</span>\n          <span>{unit}</span>\n        </div>\n      ))}\n    </div>\n  )}\n</Countdown>`}>
  <CountdownDemo />
</ComponentBlock>

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| endTime | `Date \| number` | - | 结束时间 |
| pad | `boolean` | `false` | 是否补零 |
| always | `boolean` | `false` | 倒计时结束后是否继续渲染 |
| labels | `{ days, hours, minutes, seconds }` | 中文 | 时间单位文案 |
| children | `ReactNode \| (params) => ReactNode` | - | 自定义渲染 |
| onEnd | `() => void` | - | 倒计时结束回调 |
