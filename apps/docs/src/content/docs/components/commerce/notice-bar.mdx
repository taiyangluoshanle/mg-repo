## 基本用法

<ComponentBlock cli="@mg/ui-commerce" code={`import { NoticeBar } from "@mg/ui-commerce";\n\n<NoticeBar\n  notices={[\n    { title: "🔥 限时优惠：全场满 $500 减 $50" },\n    { title: "📦 免费配送至全国各地" },\n    { title: "🎁 新用户注册即享 10% 折扣" },\n  ]}\n/>`}>
  <NoticeBarDemo />
</ComponentBlock>

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| notices | `NoticeBarItem[]` | - | 通知列表 |
| interval | `number` | `3000` | 切换间隔（ms） |
| duration | `number` | `500` | 动画时长（ms） |
| renderItem | `(item, index) => ReactNode` | - | 自定义渲染 |
