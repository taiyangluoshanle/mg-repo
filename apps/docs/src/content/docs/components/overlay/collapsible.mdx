---
title: Collapsible
description: 由按钮控制的可折叠面板。比 Accordion 更轻量，适合单个区域的展开/收起。基于 Base UI Collapsible 构建。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <CollapsibleBasicDemo />
</ComponentPreview>

```tsx
import { CollapsibleRoot, CollapsibleTrigger, CollapsiblePanel } from "@mg/ui";

export default function Demo() {
  return (
    <CollapsibleRoot>
      <CollapsibleTrigger>点击展开</CollapsibleTrigger>
      <CollapsiblePanel>
        <p>折叠内容</p>
      </CollapsiblePanel>
    </CollapsibleRoot>
  );
}
```

## API

### CollapsibleRoot

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `open` | `boolean` | — | 受控展开状态 |
| `defaultOpen` | `boolean` | `false` | 默认展开 |
| `onOpenChange` | `(open: boolean) => void` | — | 展开状态变化回调 |
| `disabled` | `boolean` | `false` | 禁用 |
