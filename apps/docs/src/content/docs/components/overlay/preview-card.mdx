---
title: Preview Card
description: 当链接被悬停时显示预览内容的弹出卡片。基于 Base UI PreviewCard 构建。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <PreviewCardBasicDemo />
</ComponentPreview>

```tsx
import {
  PreviewCardRoot,
  PreviewCardTrigger,
  PreviewCardPortal,
  PreviewCardPositioner,
  PreviewCardPopup,
  PreviewCardArrow,
} from '@mg/ui';

export default function Demo() {
  return (
    <PreviewCardRoot>
      <PreviewCardTrigger href="https://base-ui.com">Base UI 官网</PreviewCardTrigger>
      <PreviewCardPortal>
        <PreviewCardPositioner sideOffset={8}>
          <PreviewCardPopup>
            <PreviewCardArrow />
            <p>预览内容</p>
          </PreviewCardPopup>
        </PreviewCardPositioner>
      </PreviewCardPortal>
    </PreviewCardRoot>
  );
}
```

## API

### PreviewCardRoot

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `open` | `boolean` | `-` | 受控开关 |
| `defaultOpen` | `boolean` | `false` | 非受控初始打开 |
| `onOpenChange` | `(open: boolean, eventDetails) => void` | `-` | 打开/关闭时回调 |

### PreviewCardTrigger

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `delay` | `number` | 库默认值 | 悬停/聚焦后打开前的延迟（毫秒） |
| `closeDelay` | `number` | 库默认值 | 指针离开后关闭前的延迟（毫秒） |
| `href` | `string` | `-` | 作为链接时的地址 |
| `children` | `ReactNode` | `-` | 触发器内容 |
