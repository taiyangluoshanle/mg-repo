---
title: Popover
description: 弹出面板，点击触发器展示任意内容。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <PopoverBasicDemo />
</ComponentPreview>

```tsx
import { Popover, PopoverTrigger, PopoverContent } from '@mg/ui';

export default function Demo() {
  return (
    <Popover>
      <PopoverTrigger>打开面板</PopoverTrigger>
      <PopoverContent>
        <p>弹出面板内容</p>
      </PopoverContent>
    </Popover>
  );
}
```

## API

### Popover

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `open` | `boolean` | `-` | 受控开关 |
| `onOpenChange` | `(open: boolean) => void` | `-` | 状态变更 |

### PopoverContent

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `align` | `"start" \| "center" \| "end"` | `"center"` | 对齐方式 |
| `className` | `string` | `-` | 自定义类名 |
