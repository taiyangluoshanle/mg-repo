---
title: Dialog
description: 开箱即用的模态对话框组件，基于 Base UI 封装，通过 Props 配置标题、内容、Footer、尺寸、关闭控制、Loading 等，无需手动组合子组件。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

一个组件搞定所有功能，通过 `title`、`description`、`footer`、`children` 配置内容结构。

<ComponentPreview>
  <DialogBasicDemo />
</ComponentPreview>

```tsx
import { useState } from 'react';
import { Dialog, Button } from '@mg/ui';

export default function Demo() {
  const [open, setOpen] = useState(false);

  return (
    <>
      <Button onClick={() => setOpen(true)}>打开对话框</Button>
      <Dialog
        open={open}
        onOpenChange={setOpen}
        title="确认操作"
        description="此操作不可撤销，是否继续？"
        footer={
          <>
            <Button variant="default" onClick={() => setOpen(false)}>取消</Button>
            <Button onClick={() => setOpen(false)}>确认</Button>
          </>
        }
      >
        主体内容
      </Dialog>
    </>
  );
}
```

## 可滚动内容

Header 和 Footer 固定，Body 区域自动滚动。结合 `className` 设置 `max-h` 控制整体高度。

<ComponentPreview>
  <DialogScrollDemo />
</ComponentPreview>

```tsx
<Dialog
  open={open}
  onOpenChange={setOpen}
  title="用户协议"
  className="max-h-[60vh]"
  footer={...}
>
  {/* 长内容 — Body 自动出现滚动条 */}
</Dialog>
```

## API

### Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `open` | `boolean` | — | **必填**，是否显示对话框 |
| `onOpenChange` | `(open: boolean) => void` | — | **必填**，状态变化回调 |
| `title` | `ReactNode` | — | 标题（设为 `null` 隐藏标题栏） |
| `description` | `ReactNode` | — | 描述文本 |
| `children` | `ReactNode` | — | 主体内容 |
| `footer` | `ReactNode` | — | 底部操作区 |
| `size` | `"sm" \| "md" \| "lg" \| "xl" \| "full"` | `"md"` | 弹窗尺寸 |
| `showCloseButton` | `boolean` | `true` | 是否显示关闭按钮 |
| `disableBackdropClick` | `boolean` | `false` | 禁止点击遮罩关闭 |
| `disableEscapeKey` | `boolean` | `false` | 禁止 ESC 关闭 |
| `destroyOnClose` | `boolean` | `true` | 关闭后是否销毁内容 |
| `loading` | `boolean` | `false` | 加载状态（遮罩 + 阻止关闭） |
| `timeout` | `number` | — | 自动关闭延时（ms） |
| `initialFocus` | `RefObject<HTMLElement>` | — | 打开后初始聚焦的元素 |
| `className` | `string` | — | 弹窗面板类名 |
| `overlayClassName` | `string` | — | 遮罩层类名 |
| `headerClassName` | `string` | — | 标题区类名 |
| `contentClassName` | `string` | — | 主体区类名 |
| `footerClassName` | `string` | — | 底部区类名 |
| `closeButtonClassName` | `string` | — | 关闭按钮类名 |

### 可访问性

- `role="dialog"` + `aria-modal="true"`（Base UI 自动添加）
- Focus trap：焦点锁定在弹窗内
- Scroll lock：打开时禁止 body 滚动
- 关闭后焦点自动返回触发元素
- `title` → `aria-labelledby`，`description` → `aria-describedby`（Base UI 自动关联）
