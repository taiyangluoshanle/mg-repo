---
title: Context Menu
description: 右键点击或长按时弹出的菜单。基于 Base UI ContextMenu 构建，复用 Menu 子组件。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <ContextMenuBasicDemo />
</ComponentPreview>

```tsx
import {
  ContextMenuRoot,
  ContextMenuTrigger,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuSeparator,
} from '@mg/ui';

export default function Demo() {
  return (
    <ContextMenuRoot>
      <ContextMenuTrigger>
        <div>右键点击此区域</div>
      </ContextMenuTrigger>
      <DropdownMenuContent>
        <DropdownMenuItem>复制</DropdownMenuItem>
        <DropdownMenuItem>粘贴</DropdownMenuItem>
        <DropdownMenuSeparator />
        <DropdownMenuItem>删除</DropdownMenuItem>
      </DropdownMenuContent>
    </ContextMenuRoot>
  );
}
```

## API

`ContextMenuRoot` 在内部使用与下拉菜单相同的 `Menu.Root`，因此可直接使用 `DropdownMenuContent`、`DropdownMenuItem`、`DropdownMenuSeparator` 等组件渲染菜单内容（它们基于 Base UI 的 `Menu.Portal` / `Menu.Popup` / `Menu.Item`）。

### ContextMenuRoot

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `open` | `boolean` | `-` | 受控开关 |
| `defaultOpen` | `boolean` | `false` | 非受控初始打开 |
| `onOpenChange` | `(open: boolean, eventDetails) => void` | `-` | 打开/关闭时回调 |
| `children` | `ReactNode` | `-` | Trigger、Portal 等子节点 |

### ContextMenuTrigger

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `children` | `ReactNode` | `-` | 可右键或长按的触发区域 |

### 菜单内容（与 DropdownMenu 共用）

菜单项与容器参见 [DropdownMenu](/docs/components/overlay/dropdown-menu) 中的 `DropdownMenuContent`、`DropdownMenuItem`、`DropdownMenuSeparator` 等。
