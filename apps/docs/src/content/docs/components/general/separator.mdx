---
title: Separator
description: 水平或垂直分隔线，用于划分内容区域。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <SeparatorBasicDemo />
</ComponentPreview>

```tsx
import { Separator } from '@mg/ui';

export default function Demo() {
  return <Separator orientation="horizontal" />;
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | 分隔方向；水平为全宽细线，垂直为全高细线 |
| `className` | `string` | `-` | 附加类名 |
| 其他 | `React.HTMLAttributes<HTMLDivElement>` | `-` | 继承原生 `div` 属性；组件自带 `role="separator"` 与 `aria-orientation` |
