---
title: Icon
description: 图标容器，将 SVG 子元素包裹在固定尺寸的 span 中并居中显示。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <IconBasicDemo />
</ComponentPreview>

```tsx
import { Icon } from '@mg/ui';

export default function Demo() {
  return (
    <Icon size="md" aria-label="示例">
      <svg viewBox="0 0 24 24" fill="none" xmlns="http://www.w3.org/2000/svg">
        <path d="M12 2L2 7L12 12L22 7L12 2Z" stroke="currentColor" strokeWidth="2" />
      </svg>
    </Icon>
  );
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | 图标容器尺寸，内部 SVG 随容器缩放 |
| `className` | `string` | `-` | 附加到外层 `span` 的类名 |
| 其他 | `React.HTMLAttributes<HTMLSpanElement>` | `-` | 继承原生 `span` 属性（如 `aria-label`、`style` 等） |
