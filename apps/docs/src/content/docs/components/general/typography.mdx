---
title: Typography
description: 标题与正文排版组件，支持无衬线（sans）与展示（display）字体变体。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <TypographyHeadingDemo />
</ComponentPreview>

<ComponentPreview>
  <TypographyTextDemo />
</ComponentPreview>

```tsx
import { Heading, Text } from '@mg/ui';

export default function Demo() {
  return (
    <>
      <Heading level={2} variant="sans">
        二级标题
      </Heading>
      <Text as="p" variant="sans">
        正文段落示例。
      </Text>
    </>
  );
}
```

## API

### Heading

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `level` | `1 \| 2 \| 3 \| 4 \| 5 \| 6` | `1` | 语义层级，对应 `h1`–`h6` |
| `variant` | `"sans" \| "display"` | `"sans"` | 字体族变体 |
| `className` | `string` | `-` | 附加类名 |
| 其他 | `React.HTMLAttributes<HTMLHeadingElement>` | `-` | 继承对应标题元素属性 |

### Text

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `as` | `"p" \| "span"` | `"p"` | 渲染为段落或行内文本 |
| `variant` | `"sans" \| "display"` | `"sans"` | 字体族变体 |
| `className` | `string` | `-` | 附加类名 |
| 其他 | `React.HTMLAttributes<HTMLParagraphElement>` | `-` | 继承 `p`/`span` 常用属性 |
