---
title: Button 按钮
description: 触发动作或事件的交互元素。支持五种类型、图标、加载状态、幽灵模式和危险模式。
---

## 安装

```bash
pnpm add @mg/ui
```

## 按钮类型

按钮有五种类型：主按钮、次按钮、虚线按钮、文本按钮和链接按钮。

<ComponentBlock
  cli="@mg/ui"
  code={`import { Button } from "@mg/ui";

<Button variant="primary">主按钮</Button>
<Button variant="default">次按钮</Button>
<Button variant="dashed">虚线按钮</Button>
<Button variant="text">文本按钮</Button>
<Button variant="link">链接按钮</Button>`}
>
  <ButtonBasicDemo />
</ComponentBlock>

## 图标按钮

当需要在 Button 内嵌入 Icon 时，可以设置 `icon` 属性。仅设置 `icon` 不设置 `children` 时渲染为图标按钮（正方形）。

<ComponentBlock
  cli="@mg/ui"
  code={`import { Button } from "@mg/ui";

<Button variant="primary" icon={<SearchIcon />}>搜索</Button>
<Button variant="default" icon={<DownloadIcon />}>下载</Button>

{/* 图标按钮（无文字） */}
<Button variant="primary" icon={<SearchIcon />} />
<Button variant="default" icon={<DownloadIcon />} />`}
>
  <ButtonIconDemo />
</ComponentBlock>

## 按钮尺寸

通过设置 `size` 为 `large` `small` 分别把按钮设为大、小尺寸。若不设置 `size`，则尺寸为中（`middle`）。

<ComponentBlock
  cli="@mg/ui"
  code={`import { Button } from "@mg/ui";

<Button variant="primary" size="large">Large</Button>
<Button variant="primary" size="middle">Middle</Button>
<Button variant="primary" size="small">Small</Button>`}
>
  <ButtonSizesDemo />
</ComponentBlock>

## 加载中状态

添加 `loading` 属性即可让按钮处于加载状态，最后两个按钮演示点击后进入加载状态。

<ComponentBlock
  cli="@mg/ui"
  code={`import { useState } from "react";
import { Button } from "@mg/ui";

const [loading, setLoading] = useState(false);

<Button variant="primary" loading>加载中</Button>
<Button variant="primary" loading={loading} onClick={() => {
  setLoading(true);
  setTimeout(() => setLoading(false), 3000);
}}>
  点击加载
</Button>`}
>
  <ButtonLoadingDemo />
</ComponentBlock>

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `variant` | `"primary" \| "default" \| "dashed" \| "text" \| "link"` | `"default"` | 按钮类型 |
| `size` | `"large" \| "middle" \| "small"` | `"middle"` | 按钮尺寸 |
| `icon` | `ReactNode` | `-` | 设置按钮的图标 |
| `loading` | `boolean` | `false` | 加载中状态 |
| `disabled` | `boolean` | `false` | 禁用状态 |
| `ghost` | `boolean` | `false` | 幽灵模式（透明背景） |
| `danger` | `boolean` | `false` | 危险按钮 |
| `block` | `boolean` | `false` | 撑满父元素宽度 |
| `href` | `string` | `-` | 设置后渲染为 `<a>` 标签 |
| `target` | `string` | `-` | 与 `href` 配合，等同于 `<a>` 的 `target` |
| `htmlType` | `"button" \| "submit" \| "reset"` | `"button"` | 原生 button 的 type |
| `asChild` | `boolean` | `false` | 将样式代理到子元素 |
