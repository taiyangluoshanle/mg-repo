---
title: Button
description: 触发动作或事件的交互元素。支持多种变体、尺寸和组合模式。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <ButtonBasicDemo />
</ComponentPreview>

```tsx
import { Button } from '@mg/ui';

export default function Demo() {
  return <Button variant="primary">点击我</Button>;
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `variant` | `"primary" \| "secondary" \| "ghost" \| "destructive" \| "outline"` | `"primary"` | 按钮变体 |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | 按钮尺寸 |
| `asChild` | `boolean` | `false` | 将样式代理到子元素 |
| `disabled` | `boolean` | `false` | 禁用状态 |
