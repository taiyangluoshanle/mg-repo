---
title: Button
description: 触发动作或事件的交互元素。支持多种变体、尺寸和组合模式。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <ButtonBasicDemo />
</ComponentPreview>

```tsx
import { Button } from '@mg/ui';

export default function Demo() {
  return <Button variant="primary">点击我</Button>;
}
```

## 变体

所有变体：`primary`、`secondary`、`outline`、`ghost`、`destructive`。

<ComponentPreview>
  <ButtonVariantsDemo />
</ComponentPreview>

```tsx
import { Button } from '@mg/ui';

export default function Demo() {
  return (
    <div className="flex flex-wrap items-center gap-3">
      <Button variant="primary">Primary</Button>
      <Button variant="secondary">Secondary</Button>
      <Button variant="outline">Outline</Button>
      <Button variant="ghost">Ghost</Button>
      <Button variant="destructive">Destructive</Button>
    </div>
  );
}
```

## 尺寸

三种尺寸：`sm`、`md`（默认）、`lg`。

<ComponentPreview>
  <ButtonSizesDemo />
</ComponentPreview>

```tsx
import { Button } from '@mg/ui';

export default function Demo() {
  return (
    <div className="flex items-center gap-3">
      <Button size="sm">Small</Button>
      <Button size="md">Medium</Button>
      <Button size="lg">Large</Button>
    </div>
  );
}
```

## 禁用状态

<ComponentPreview>
  <ButtonDisabledDemo />
</ComponentPreview>

```tsx
<Button disabled>Disabled</Button>
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `variant` | `"primary" \| "secondary" \| "ghost" \| "destructive" \| "outline"` | `"primary"` | 按钮变体 |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | 按钮尺寸 |
| `asChild` | `boolean` | `false` | 将样式代理到子元素 |
| `disabled` | `boolean` | `false` | 禁用状态 |
