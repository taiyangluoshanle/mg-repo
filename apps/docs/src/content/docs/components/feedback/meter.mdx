---
title: Meter
description: 在已知范围内显示标量值的图形指示器。适用于展示存储空间、电量等。基于 Base UI Meter 构建。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <MeterBasicDemo />
</ComponentPreview>

```tsx
import { MeterRoot, MeterTrack, MeterIndicator, MeterLabel, MeterValue } from "@mg/ui";

export default function Demo() {
  return (
    <MeterRoot value={72}>
      <div className="flex items-center justify-between">
        <MeterLabel>存储空间</MeterLabel>
        <MeterValue />
      </div>
      <MeterTrack>
        <MeterIndicator />
      </MeterTrack>
    </MeterRoot>
  );
}
```

## API

### MeterRoot

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `value` | `number` | — | 当前值 |
| `min` | `number` | `0` | 最小值 |
| `max` | `number` | `100` | 最大值 |
