---
title: Alert
description: 用于页面内提示、结果反馈等场景；左侧强调色条区分信息、成功、警告与错误。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

```tsx
import { Alert } from '@mg/ui';

export default function Demo() {
  return (
    <div className="flex flex-col gap-3">
      <Alert variant="info" title="提示">
        这是一条一般说明信息。
      </Alert>
      <Alert variant="success" title="成功">
        操作已完成。
      </Alert>
      <Alert variant="warning" title="注意">
        请检查输入后再提交。
      </Alert>
      <Alert variant="error" title="错误" icon={<span aria-hidden>!</span>}>
        请求失败，请重试。
      </Alert>
    </div>
  );
}
```

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `variant` | `"info" \| "success" \| "warning" \| "error"` | `"info"` | 语义变体 |
| `title` | `string` | — | 可选标题 |
| `icon` | `ReactNode` | — | 标题前图标 |
| `children` | `ReactNode` | — | 正文内容 |
| `className` | `string` | — | 附加类名 |
| 其他 | `HTMLAttributes<HTMLDivElement>` | — | 容器带 `role="alert"` |
