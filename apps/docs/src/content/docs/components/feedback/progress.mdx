---
title: Progress
description: 进度条，展示 0–100 的完成比例；支持成功/错误语义色与多种轨道高度。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <ProgressBasicDemo />
</ComponentPreview>

```tsx
import { Progress } from '@mg/ui';

export default function Demo() {
  return (
    <div className="flex w-full max-w-md flex-col gap-4">
      <Progress value={35} />
      <Progress value={80} variant="success" size="sm" />
      <Progress value={20} variant="error" size="lg" />
    </div>
  );
}
```

`value` 会被限制在 `0`–`100` 区间内；组件具备 `role="progressbar"` 与对应 `aria-*` 属性。

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `value` | `number` | （必填） | 当前进度 0–100 |
| `variant` | `"default" \| "success" \| "error"` | `"default"` | 指示条颜色语义 |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | 轨道高度 |
| `className` | `string` | — | 附加在轨道容器上 |
| 其他 | `HTMLAttributes<HTMLDivElement>`（不含 `children`） | — | 外层 `div` 属性 |
