---
title: Spinner
description: 旋转加载指示器，用于异步等待场景；提供三种像素尺寸。
---

## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <SpinnerSizesDemo />
</ComponentPreview>

```tsx
import { Spinner } from '@mg/ui';

export default function Demo() {
  return (
    <div className="flex items-center gap-4">
      <Spinner size="sm" />
      <Spinner size="md" />
      <Spinner size="lg" />
    </div>
  );
}
```

图标带有 `aria-hidden`，请在外层配合可见文案或 `aria-label` 说明加载状态，以满足可访问性。

## API

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | 图标边长（16 / 24 / 32 px） |
| `className` | `string` | — | 附加类名 |
| 其他 | `SVGProps<SVGSVGElement>`（不含 `width`/`height`） | — | 宽高由 `size` 控制 |
