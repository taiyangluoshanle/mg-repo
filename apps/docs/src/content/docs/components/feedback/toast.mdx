## 安装

```bash
pnpm add @mg/ui
```

## 基础用法

<ComponentPreview>
  <ToastBasicDemo />
</ComponentPreview>

在应用根组件（如 `layout.tsx`）外层使用 `ToastProvider`（内部已挂载 `Toaster`，无需再单独放置）。

```tsx
'use client';

import { ToastProvider } from '@mg/ui';

export function Providers({ children }: { children: React.ReactNode }) {
  return <ToastProvider>{children}</ToastProvider>;
}
```

在任意子组件中调用：

```tsx
'use client';

import { Button, useToast } from '@mg/ui';

export function Demo() {
  const { toast } = useToast();

  return (
    <Button
      variant="primary"
      onClick={() =>
        toast({
          title: '已保存',
          description: '您的更改已同步。',
          variant: 'success',
          duration: 4000,
        })
      }
    >
      显示 Toast
    </Button>
  );
}
```

`toast()` 的选项类型为 `ToastOptions`：`title` 必填，其余可选。

## API

### ToastProvider

| 属性 | 类型 | 说明 |
|------|------|------|
| `children` | `ReactNode` | 子树；内部会渲染通知栈与 `Toaster` |

### useToast()

| 返回值 | 类型 | 说明 |
|--------|------|------|
| `toast` | `(options: ToastOptions) => void` | 推入一条通知 |

### ToastOptions

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `title` | `ReactNode` | （必填） | 标题 |
| `description` | `ReactNode` | — | 副标题/详情 |
| `variant` | `"default" \| "success" \| "error"` | `"default"` | 样式变体 |
| `duration` | `number` | `5000` | 自动关闭毫秒数 |

### Toaster

通常由 `ToastProvider` 自动包含；若需自定义挂载位置，可在已提供同一上下文的前提下单独使用（一般不必）。
