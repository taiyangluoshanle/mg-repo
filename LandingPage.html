Button

触发动作或事件的交互元素。基于 Base UI useButton hook 构建，支持多种变体、尺寸和 asChild 组合模式。

预览
代码

安装

bash
pnpm add @mg/ui

基础用法

tsx
import { Button } from '@mg/ui'; const Demo = () => ( <Button variant="primary" size="md"> 点击我 </Button> );

变体 Variants

primary
secondary
ghost
destructive

尺寸 Sizes

API

属性 类型 默认值 说明
variant "primary" | "secondary" | "ghost" | "destructive" "primary" 按钮视觉变体
size "sm" | "md" | "lg" "md" 按钮尺寸
asChild boolean false 将样式和行为代理到子元素
disabled boolean false 禁用状态

无障碍 Accessibility

✓ 支持键盘操作：Enter 和 Space 触发点击
✓ 自动设置 role="button" 当使用非 button 元素时
✓ 禁用时自动设置 aria-disabled="true"
